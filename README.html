Inner Root

Indian Heritage & Wellness Platform

Inner Root is a next-generation immersive platform that blends the exploration of Indian cultural heritage with AI-driven spiritual and emotional guidance. It provides a cinematic and emotionally resonant digital experience for users to connect with their roots and improve their inner well-being.

🌟 Key Features

🏛️ Heritage Exploration

  • Virtual Tours: Immersive exploration of historical landmarks.
  • Cultural Database: Deep dives into Traditions and Festivals.
  • Artifacts: Digital access to cultural historical records.

🧘 Wellness & Wisdom

  • Wellness Module: Meditation, Yoga, and Pranayama.
  • Daily Wisdom: AI-driven spiritual insights.
  • Mood Tracking: Emotional wellness journaling.

🛠️ Tech Stack

Backend

Spring Boot 3.5.10 Java 21 Supabase/PostgreSQL JWT & OAuth2 Maven

Frontend

React 19 (Vite) Tailwind CSS Framer Motion React Router 7 Lucide Icons

📂 Project Structure

e:/v2 inner/
├── inner-root-backend/    # Spring Boot application
├── react-project/         # React frontend application
└── README.html            # Project documentation (HTML)

🚀 Getting Started

Backend Setup

cd inner-root-backend
./mvnw spring-boot:run

Frontend Setup

cd react-project
npm install
npm run dev
