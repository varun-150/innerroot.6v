# Inner Root - Indian Heritage Platform (Production Ready)

Welcome to the **Ancient Future** digital ecosystem. This repository contains both the **React Frontend** and the **Spring Boot Java Backend**, fully optimized for performance, security, and scalability.

## 🚀 Recent Deployment Updates
- **Lazy Loading**: Implemented code splitting for all routes to significantly reduce initial bundle size.
- **Security Scrubbing**: All sensitive API keys and database credentials have been removed from the source code.
- **Environment Driven**: The application is now fully configured for remote deployment using Environment Variables.

## 📁 Repository Structure
- `/inner-root-backend`: Spring Boot (Java) application.
- `/react-project`: Vite + React (JavaScript) application.
- `/scripts`: Automation utilities.
